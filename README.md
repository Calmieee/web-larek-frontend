# Проектная работа "Веб-ларек"

Стек: HTML, SCSS, TS, Webpack

Структура проекта:
- src/ — исходные файлы проекта
- src/components/ — папка с JS компонентами
- src/components/base/ — папка с базовым кодом

Важные файлы/папки:
- src/pages/index.html — HTML-файл главной страницы
- src/types/ — папка с типами
- src/index.ts — точка входа приложения
- src/styles/styles.scss — корневой файл стилей
- src/utils/constants.ts — файл с константами
- src/utils/utils.ts — файл с утилитами

## Установка и запуск
Для установки и запуска проекта необходимо выполнить команды

```
npm install
npm run start
```

или

```
yarn
yarn start
```
## Сборка

```
npm run build
```

или

```
yarn build
```

### События
---
1. **Клик по карточке товара** - обработчик передаёт данные объекта Card в CardView, где далее формируется контент и открывается модальное окно с этим контентом.

2. **Клик по кнопке закрытия попапа** - обработчик вызовет метод close() объекта ModalView

3. **Клик по кнопке "В корзину" внутри попапа** - обработчик вызовет объект AppState, где методом appendItemInBasket товар добавится свойство в  AppState.Basket.

    * Далее сработает ещё один обработчик, который прибавляет единицу товара в BasketItemView.index для последующего отображения счётчика продуктов у иконки корзины.

4. **Клик по корзине** - обработчик вызовет методы getBasket() и getTotalPrice() объекта AppState, передаст данные в объект BasketView. Далее вызовется модальное окно для отображения корзины(ModalView)

    * **Клик по кнопке закрытия попапа** - обработчик вызовет метод close() объекта ModalView
    * **Клик по кнопке "Оформить"** - обработчик откроет модальное окно с переданными данными объекта OrderView и от его родительского класса FormView

5. События внутри оформления заказа:
        
    * **Клик по кнопке закрытия попапа** - обработчик вызовет метод close() объекта ModalView
    * **Клик по кнопкам выбора способа оплаты** - обработчик запишет(перепишет, если поменялся выбор) состояние методом в OrderView и выделит в интерфейсе выбор.
    * **Ввод данных в поле адреса доставки** -обработчик вызовет AppState.validateOrder() для валидации формы; состояния валидации и ошибки будет передавать в FormView для отображения
    * **Клик по кнопке "Далее"** - обработчик запишет вводимые данные в AppState.OrderData методом setOrderData(data) и отобразить следующее окно заказа для заполнения

    * **Ввод данных в инпут поля** - аналогично событию "Ввод данных в поле адреса доставки"

    * **Клик по кнопке "Оплатить"** - обработчик запишет вводимые данные в AppState.OrderData методом setOrderData(data). Далее вызовет Api.Post(data), где методом "POST" данные заказа отправятся на сервер. ModalView отобразить финальное окно.

    * **Клик по кнопке "за новыми покупками"** - обработчик вызовет AppState.clearOrder() для очищения данных заказа и закроет модальное окно.

    * **Клик по кнопке закрытия попапа** - обработчик вызовет метод close() объекта ModalView
